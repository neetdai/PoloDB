
# PoloDB Rust Manual

# Installation

```toml
[dependencies]
polodb_core = "0.4.3"
polodb_bson = "0.4.3"
```
# 使用

## Open a database

```rust
let mut db = Database::open(db_path.as_path().to_str().unwrap())?;
```

## Use Collection

Create
```rust
let mut collection = db.create_collection("test")?;
```

Get
```rust
let mut collection = db.collection("test")?;
```

## Count

```rust
let count = collection.count()?;
```

## Insert

```rust
let new_doc = mk_document! {
                "name": "Vincent Chan",
            };
collection.insert(Rc::new(new_doc)).unwrap();
```

## Find

```rust
let result = collection.find(
            Some(mk_document! {
                "content": "3",
            }.borrow())
        )?;
```

## Update

```
collection.update(...);
```

## Delete

```
collection.delete(...);
collection.delete_all();
```

## Transaction

```rust
db.start_transaction(None).unwrap();
let mut collection = db.create_collection("test").unwrap();

// something

db.commit().unwrap()
```
